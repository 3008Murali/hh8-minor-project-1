# hh8-minor-project-1
Ransomware Simulation(Encrypts a test folder to show impact)
Project description 
The current project is an educational simulation of the ransomware attack made using Python programming.
This shows how the ransomware encrypts user files in order to illustrate its effects and generate awareness about cybersecurity threats.
It will only function with the test directory set by the user, holding the dummy files.
It does not affect the system files. There is a decryption script included. The project at hand is strictly for educational purposes.

Tools used
Python 3.x
Cryptography Library (Fernet)
VS Code
